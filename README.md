# Financial Strain and Mortality Among US Adults Aged 50 Years or Older

Replication code for: Tharp, D. (2026). Financial strain and all-cause mortality among US adults aged 50 years and older: age-dependent vulnerability and attenuation by socioeconomic resources.

## Study Design

- **Design:** Prospective cohort study
- **Data Source:** Health and Retirement Study (HRS), Wave 8 (2006) through Wave 16 (2022)
- **Sample:** 7,219 adults aged 50+ years who completed the Leave-Behind Questionnaire
- **Follow-up:** Up to 16 years (mean 11.5 years)
- **Exposure:** Ongoing financial strain (binary: high vs. low/no strain)
- **Primary outcome:** All-cause mortality (3,466 deaths, 48.0%)
- **Secondary outcome:** Incident heart disease among participants free of CVD at baseline

## Requirements

- **Python 3.13** with packages listed in `requirements.txt`
- **Stata MP 18+** for survey-weighted Cox models (script 09) and competing risks (script 07)
- **Data:** RAND HRS Longitudinal File v1 (2022); HRS 2006 Biomarker data and Cross-wave CRP data (restricted access; see Data Access below)

### Setup

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Analysis Scripts

Scripts are numbered in execution order. Run all with `python run_all.py` or verify outputs with `python run_all.py --verify`.

| Script | Purpose |
|---|---|
| `model_specs.py` | Centralised covariate definitions (imported by analysis scripts) |
| `01_primary_analysis.py` | Primary Cox models, sample construction, all main tables |
| `01b_export_survey_data.py` | Merge HRS survey design variables (strata, PSU) onto analytic sample |
| `02_figures.py` | All publication figures (KM curves, forest plot, PS overlap, DAG) |
| `03_secondary_analysis.py` | Supplementary analyses, Stata export, participant flow |
| `04_flow_diagram.py` | Participant flow diagram |
| `05_sensitivity_weighted.py` | PH diagnostics, weighted estimates, LBQ selection bias |
| `06_sensitivity_clustering.py` | Standardised risks, household clustering |
| `07_competing_risks_heart_disease.do` | Fine-Gray competing risks for incident heart disease (Stata) |
| `08_exploratory_reviewer_analyses.py` | Lexis expansion, PS-matched age strata |
| `09_survey_design_cox.do` | Survey-weighted Cox models with full HRS design (Stata `svy: stcox`) |
| `10_biomarker_analysis.py` | Exploratory CRP/biomarker analyses (restricted data required) |

## Data Access

The Health and Retirement Study data are publicly available but require free registration:

- **RAND HRS Longitudinal File:** https://www.rand.org/well-being/social-and-behavioral-policy/centers/aging/dataprod/hrs-data.html
- **HRS main site:** https://hrs.isr.umich.edu/

The biomarker analyses (script 10) require restricted-access data available through an HRS data use agreement:

- **2006 Biomarker data** (dried blood spot CRP, HbA1c, cholesterol, cystatin C)
- **Cross-wave CRP data** (2006-2016)

Place the data files as follows:

- `randhrs1992_2022v1_STATA/randhrs1992_2022v1.dta` — RAND HRS Longitudinal File
- `Special Access Data/biomkr06/biomk06bl_r.sav` — 2006 Biomarker data
- `Special Access Data/dbs_crp/HRS_CRP_XWAVE.dta` — Cross-wave CRP data

## Output

Running the scripts produces all tables (CSV) in `output/tables/` and all figures (PDF and PNG) in `output/figures/`. Output files are not included in this repository; they are generated by running the analysis scripts.

## Statistical Methods

- Cox proportional hazards regression (sequential covariate adjustment)
- Survey-weighted estimates (HRS LBQ respondent weights, Stata `svy: stcox`)
- Propensity score matching (1:1 nearest-neighbour)
- Fine-Gray subdistribution hazards for competing risks (Stata `stcrreg`)
- E-value sensitivity analysis for unmeasured confounding
- Dose-response analysis (4-level ordinal exposure)
- Age effect modification (binary and continuous interaction)
- Exploratory CRP/biomarker associations (age-stratified OLS)
- Lexis expansion with time-varying age-65 splitting
- Covariate-standardised absolute risks
- Household-clustered robust standard errors
